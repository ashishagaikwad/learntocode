Finding HTML Elements

Following are some waya:

  • Finding HTML elements by id
  • Finding HTML elements by tag name
  • Finding HTML elements by class name
  • Finding HTML elements by CSS selectors
  • Finding HTML elements by HTML object collections
------------------------------------

Finding html elements by id

document.getElementById()
This is used to get elements by using id attribute associated with it
------------------------------------

------------------------------------

Finding html elements by TagName

document.getElementsByTagName()
This is used to get elements by using tags associated with it. We get an array of values for all tags..which we have to access using index
--------------------------------------

Finding Elements by Class Name

document.getElementByClassName
If you want to find all HTML elements with the same class name, use getElementsByClassName().

---------------------------------

Finding Elements by CSS Selectors

document.querySelectorAll()
If you want to find all HTML elements that matches a specified CSS selector (id, class names, types, attributes, values of attributes, etc), use the querySelectorAll() method.
