Window Size

Three different properties can be used to determine the size of the browser window
1. window.innerHeight / window.innerWidth                                           // for IE/Chrome/safari/Opera
2. document.documentElement.clientHeight / document.documentElement.clientWidth     // for IE 8,7,6,5
3. document.body.clientHeight / document.body.clientWidth							// for IE 8,7,6,5

Note: Values will differ for different browsers
