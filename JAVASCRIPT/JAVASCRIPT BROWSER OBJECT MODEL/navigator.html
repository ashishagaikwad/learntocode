Window Navigator Object



The window.navigator object contains information about the visitors browser

Can be written without the windows prefix


PROPERTIES OF THE NAVIGATOR OBJECT

1.  appName         //Returns the name of the browser
2.  appCodeName     //Returns the name of the browser
3.  platform        //Returns the browsers platform ( Operating System )
4.  cookieEnabled   //Returns true if cookie is enabled otherwise false
5.  javaEnabled     //Returns true if java is enabled otherwise false
6.  product         //Returns the engine name of the browser
7.  appVersion      //Returns the version information about the browser
8.  userAgent       //Also returns the version information about the browser
9.  language        //Returns the browser language


Note :
1. chrome ,firefox and safari return appName "Netscape"
2. IE,Chrome,Firefox,Safari and Opera all return appCodeName "Mozilla" 
3. Browser version data can be misleading since browser can change them to avoid site tests
