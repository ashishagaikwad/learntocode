JavaScript Data Types

  • JavaScript variables can hold many data types: numbers,strings, arrays, objects and more.
  • JavaScript evaluates expressions from left to right. Different sequences can produce different results: for Example : var a = 16 + 4 + "ashish"; document.write(a); //This will give 20ashish as output while var b = "ashish" + 16 + 4; document.write(b); //This will give ashish164 as output
